# Collaboration App Hi-Fi Prototype

## Overview
This project presents a high-fidelity prototype of a collaboration and teamwork application designed to improve task allocation, communication, and productivity within teams. The system integrates AI-assisted features to enhance user experience and efficiency.

## Figma Prototype
**Access the interactive prototype here:**  
https://www.figma.com/design/Roip2Ptf0TuOMMb1z3DvaB/INFO2222?node-id=0-1&p=f&t=9yLjH38fWNiq2IJy-0

## Functional Features
- **Dashboard** with task tracking and progress indicators  
- **AI task delegator** for assigning tasks based on skills  
- **Chat system** with AI-generated summaries  
- **Contribution analytics** to track team performance  
- **File upload and management system**

## How to Use
1. Open the Figma link above  
2. Click the **"Present" (Run)** button in the top-right corner  
3. Start from the login screen:  
   - Click **"Log In"** if you have an account  
   - Click **"Create Account"** to register  
4. After logging in, you will be directed to the profile/dashboard page  
5. Navigate through the system by clicking different features such as:  
   - Dashboard  
   - Task Delegator  
   - Chat  
   - File Management  

## Technologies Used
- **Figma (UI/UX Design)**

## Interactive Features
- Navigation between login, signup, and all available pages  
- Clickable buttons for switching between different system features  
- Simulated user flow across multiple screens  

## Limitations / TODO
- No backend functionality implemented  
- Chat is not real-time (no Socket.io integration)  
- No database storage  
- AI features are simulated only  
- No security or authentication system implemented  
- Users cannot switch between multiple projects (the dashboard is static for one project only)  

## AI Usage (Appendix)
AI tools were used to assist in generating UI design ideas and feature concepts. All outputs were reviewed and refined before inclusion in the prototype.
